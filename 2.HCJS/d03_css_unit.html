CSS3 Units Demo

CSS has several different units for expressing a length.
Many CSS properties take "length" values, such as width, margin, padding, font-size, etc.
Length is a number followed by a length unit, such as 10px, 2em, etc.

Example px:
this text has default font-size this text has font-size is 28px this text has font-size is 18px

Example em:
this text has default font-size this text has font-size is 1em this text has font-size is 2em this text has font-size is 1.3em

Example ex:
this text has default font-size this text has font-size is 1ex this text has font-size is 2ex this text has font-size is 3ex

Example rem:
this text has default font-size this text has font-size is 1rem this text has font-size is 2rem this text has font-size is 1.3rem

Example vw:

div { width : 50%; }
img {width: 50%;}

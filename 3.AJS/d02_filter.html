filter angularjs demo

Format a number

{{num}} | number = {{num |number}}
{{num}} | number:2 = {{num |number:2}}
{{num}} | number:4 = {{num |number:4}}

Format a currency

{{num}} | currency = {{num |currency}}
{{num}} | currency:'':0 = {{num |currency:'':0}}
{{num}} | currency:0:0 = {{num |currency:0:0}}
{{num}} | currency:'VND ':0 = {{num |currency:'VND ':0}}
{{num}} | currency:'EUR' = {{num |currency:'EUR '}}
